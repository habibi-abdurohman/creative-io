# ⚡ Creative.io - The Ultimate Creator Workspace

Creative.io adalah sebuah aplikasi web produktivitas dan kolaborasi modern yang dirancang khusus untuk kreator konten, penulis naskah, dan tim kreatif. Aplikasi ini bertindak sebagai pusat kendali (*all-in-one workspace*) untuk menyusun ide, menulis naskah, mencatat inspirasi spontan, hingga berkolaborasi dalam waktu nyata (*real-time*).

## 🚀 Fitur Utama

* **Bento Dashboard:** Launcher cerdas bergaya *asymmetrical grid* yang bersih, proporsional, dan bebas distraksi untuk navigasi cepat.
* **Script Writer:** Ruang kerja luas dan imersif untuk menyusun, mengedit, dan memanajemen draf naskah secara profesional.
* **Idea Board:** Bank data visual untuk menyimpan tautan referensi konten dan kerangka awal ide.
* **Quick Notes:** Buku catatan ringan tanpa format rumit untuk menangkap inspirasi yang datang tiba-tiba.
* **Collab Hub (Real-time):** Ruang kolaborasi tim interaktif dengan fitur *Live Presence* (Facepile) dan pembaruan data waktu nyata.
* **Financial Calculator:** Alat simulasi matematis untuk menghitung cicilan, bunga, dan tenor pembiayaan.
* **Trash Recovery:** Sistem *soft-delete* (Time-To-Live) yang mengamankan data terhapus selama 30 hari sebelum dimusnahkan permanen.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables & Glassmorphism UI), Vanilla JavaScript (ES6+).
* **Backend & Database:** Firebase Authentication & Cloud Firestore (NoSQL).
* **Architecture:** Progressive Web App (PWA) Ready, Mobile-First Responsive Design.
